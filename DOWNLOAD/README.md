# Training and Important Notes
---

## Training:
1. The downloaded file is located at  
``` {your repo}/DOWNLOAD/{`name file` or `custom name`} ```
2. In the Download Links section, click the Download button.

---

## Notes:

- Each downloaded file includes a checksum file (SHA256SUMS) to verify its integrity.
- If you need to delete downloaded files, select the "Purge DOWNLOAD directory" option in the Actions tab to clean the entire DOWNLOAD folder.
