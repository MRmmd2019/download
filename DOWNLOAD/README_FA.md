# آموزش و نکات مهم
---

## آموزش:
1. فایل دانلود شده در مسیر  
``` {your repo}/DOWNLOAD/{`name file` or `custom name`} ```
قرار دارد.
2. در بخش Download Links روی دکمه Download کلیک کنید.

---

## نکات:

- هر فایل دانلودی شامل یک فایل چک سام (SHA256SUMS) است که سلامت آن را تایید می‌کند.
- در صورتی که نیاز به حذف فایل‌های دانلودی بود، در بخش Actions گزینه Purge DOWNLOAD directory را انتخاب کنید تا تمام فولدر DOWNLOAD پاک‌سازی شود.
